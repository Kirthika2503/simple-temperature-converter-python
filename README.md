# Temperature Converter (Python)

A simple Python program that converts temperature between **Celsius and Fahrenheit**.  
The program accepts both **uppercase and lowercase** unit inputs for better user experience.

---

## Features
- Convert **Celsius → Fahrenheit**
- Convert **Fahrenheit → Celsius**
- Accepts `C/c` and `F/f`
- Rounds the result to 2 decimal places
- Handles invalid unit input gracefully
- Beginner-friendly Python code

---

## How It Works
1. User enters the temperature value.
2. User selects the unit:
   - `C` or `c` for Celsius
   - `F` or `f` for Fahrenheit
3. The program converts the temperature and displays the result.
